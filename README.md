# 💵 해외주식 배당 설계기

목표 월배당 ↔ 필요 투자금을 **양방향**으로 계산하고, **고배당 ETF**와 **커버드콜** 두 가지 시나리오를 따로 짜서 비교하는 단일 HTML 웹앱.

## 기능
- **두 가지 계산 모드**
  - 🎯 목표 월배당 → 필요 투자금 (예: "월 200만원 받으려면 얼마?")
  - 💰 투자금 → 예상 월배당 (예: "1억 넣으면 매달 얼마?")
- **포트폴리오 탭 2개** — 고배당 ETF(안정·저변동) / 커버드콜(고수익 월배당)
- **미국 배당세 15% 반영** — 세후 실수령액 기준 계산 (토글)
- **종목별 시세·배당률·배분 직접 수정** — 수정값은 브라우저에 자동 저장
- 원/달러 환율 입력, 균등 배분·기본값 복원, 가중 평균 배당률 표시

## 사용법
별도 설치 없이 브라우저로 `index.html`만 열면 됩니다. (또는 GitHub Pages 주소 접속)

## ⚠️ 주의
시세·배당률은 참고용 스냅샷이며 실제 값과 다를 수 있습니다(표에서 직접 수정 가능). 본 도구는 **투자 권유가 아닌 계산·시뮬레이션 보조**입니다. 커버드콜 ETF는 고배당 대신 주가 상승 제한·원금 손실·배당 변동 위험이 있습니다. 투자 책임은 본인에게 있습니다.
